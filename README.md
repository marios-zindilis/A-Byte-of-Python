In 2009, a group of members of the Greek Ubuntu Community translated the book "A Byte of Python" from
English to Greek. At the time, the book was specific to Python 2, which has not been since 2020. The
translation was published at the -now defunct- Greek Ubuntu community wiki, at address
`http://wiki.ubuntu-gr.org/byte-of-python-el`. This repository contains the code for the scripts from
that book.

The latest version of "A Byte of Python" in English can be found at
[python.swaroopch.com](https://python.swaroopch.com/).
